# Tickled Pink Fabric Shop - Web Project

Hi! Welcome to my first web development project. This is a practice website I am building for a cozy fabric store called "Tickled Pink Fabric Shop." I started this project by sketching ideas on paper and turned those drawings into a real, working webpage using HTML and CSS.

## What This Project Is
This is the homepage for a local fabric shop. The site is designed to help crafters and sewing fans browse beautiful fabrics, check out upcoming crafting classes, and watch sewing tutorials online.

##  Why I Built This
I created this website to learn the basics of building websites from scratch. My goals for this project were:
* To learn how to structure a webpage correctly using HTML tags.
* To make the site look clean and pretty using a soft pink and berry color palette.
* To practice using Flexbox so the page looks great on both small phones and wide computer screens.

## Features
* **Mobile Friendly:** The navigation menu stacks neatly on phones but stretches out into a horizontal bar on computers.
* **Photo Grid:** A flexible four-column photo row at the bottom ready to showcase fabric patterns.
* **Contact Box:** A centered contact area in the footer so customers can easily find the store's email.

##  File Structure
* `index.html` - The main structure and content of the homepage.
* `style.css` - All the colors, fonts, margins, and layouts.
* `README.md` - This instruction file.

##  How to Open the Site
You don't need any special software to run this project! 
1. Download all the files into a folder on your computer.
2. Double-click the `index.html` file.
3. It will instantly open up in your favorite web browser (like Chrome, Safari, or Edge).
